Adds citations to content dragged or copy/pasted off your website using the HTML5 drag and drop API.

== Description ==

The CiteDrag plugin automatically includes the CiteDrag JavaScript library on every page of your website. The CiteDrag JavaScript library adds citations to content dragged or copy/pasted off a website. It only works for browsers that support the HTML5 drag and drop API.

== Installation ==

1. After downloading the CiteDrag plugin, unpack and upload the file to the wp-content/plugins folder on your blog. Make sure to leave the directory structure of the archive intact so that all of the CiteDrag files are located in 'wp-content/plugins/citedrag/'

2. You will need to activate the CiteDrag plugin in order for it to take effect. Go to the Plugins tab and find CiteDrag in the list and click **Activate**.

== Frequently Asked Questions ==

= What browsers support the HTML5 drag and drop API? =

As of April 27, 2009, only Firefox 3.1/3.5 supports the HTML5 drag and drop API. [After WebKit bug 23695][1] is fixed, CiteDrag will also work on WebKit-based browsers like Safari and Google Chrome.


  [1]: https://bugs.webkit.org/show_bug.cgi?id=23695
